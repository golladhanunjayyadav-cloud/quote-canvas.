# 🪶 Quote Canvas

A dynamic daily quotes web app that changes its background image based on the quote's category — built with vanilla HTML, CSS, and JavaScript, featuring a sleek glassmorphism UI.

🔗 **Live Demo:**https://golladhanunjayyadav-cloud.github.io/quote-canvas./

---

## ✨ Features

- 📜 100+ handpicked quotes across 10 categories (motivational, life, success, love, wisdom, courage, dreams, happiness, perseverance, friendship)
- 🎨 **Dynamic backgrounds** — the background image smoothly fades and changes to match each quote's category
- 💎 Glassmorphism-style card UI with blur and shadow effects
- ❤️ **Favorite** quotes — saved locally so they persist between visits
- 📋 **Copy** quote text + author to clipboard in one click
- 🔗 **Share** directly via the Web Share API (or WhatsApp as a fallback)
- ⬅️ ➡️ Navigate quotes with Previous / Next controls

## 🛠️ Built With

- HTML5
- CSS3 (Glassmorphism, transitions, flexbox)
- Vanilla JavaScript (DOM manipulation, localStorage, Clipboard API, Web Share API)
